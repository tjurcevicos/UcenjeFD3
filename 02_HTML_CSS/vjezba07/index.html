Pokretanje čitača zaslona

  • WIN: Pritisniti na tipkovnici:
    Ctrl + Windows tipka + Enter

    Command (⌘) + F5
    (na nekim Macovima držati i fn + Cmd + F5) za pokretanje VoiceOver.
